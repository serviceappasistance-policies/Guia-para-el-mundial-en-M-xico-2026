## 📲 Descargar e instalar la app

1. Ve a la sección **Releases** (arriba, a la derecha)
2. Deslizate hasta abajo y haz clic en el archivo `.apk`
3. Cuando termine la descarga, abre el archivo
4. Si es la primera vez, permite la instalación desde "Orígenes desconocidos"
5. la primera vez te pedira permiso para poder usar GPS mientras enstas usando la app.
6. Tener en cuenta que ningun tipo de dato es recopilado.
7. Toca **Instalar** y listo

> ⚠️ Necesitas cualquier dispositivo con S.O. Android
